## 二、关键文件说明

### 2.1 配置文件

| 文件路径 | 作用 |
|---------|------|
| `pom.xml` | Maven 依赖管理，定义所有模块和依赖版本 |
| `ruoyi-ui/.env.development` | 前端开发环境配置（API 地址等）|
| `ruoyi-ui/vue.config.js` | 前端构建配置（代理、打包等）|
| `ruoyi-gateway/src/main/resources/bootstrap.yml` | 网关 Nacos 配置 |
| `ruoyi-auth/src/main/resources/bootstrap.yml` | 认证中心 Nacos 配置 |
| `ruoyi-modules/ruoyi-system/src/main/resources/bootstrap.yml` | 系统模块 Nacos 配置 |

### 2.2 启动类

| 文件路径 | 作用 |
|---------|------|
| `ruoyi-gateway/src/main/java/com/ruoyi/gateway/RuoYiGatewayApplication.java` | 启动网关服务 |
| `ruoyi-auth/src/main/java/com/ruoyi/auth/RuoYiAuthApplication.java` | 启动认证中心 |
| `ruoyi-modules/ruoyi-system/src/main/java/com/ruoyi/system/RuoYiSystemApplication.java` | 启动系统模块 |
| `ruoyi-modules/ruoyi-gen/src/main/java/com/ruoyi/gen/RuoYiGenApplication.java` | 启动代码生成 |
| `ruoyi-modules/ruoyi-job/src/main/java/com/ruoyi/job/RuoYiJobApplication.java` | 启动定时任务 |
| `ruoyi-modules/ruoyi-file/src/main/java/com/ruoyi/file/RuoYiFileApplication.java` | 启动文件服务 |

### 2.3 前端关键文件

| 文件路径 | 作用 |
|---------|------|
| `ruoyi-ui/src/main.js` | 前端入口文件 |
| `ruoyi-ui/src/App.vue` | 根组件 |
| `ruoyi-ui/src/permission.js` | 路由权限控制 |
| `ruoyi-ui/src/router/index.js` | 路由配置 |
| `ruoyi-ui/src/store/` | Vuex 状态管理 |
| `ruoyi-ui/src/api/` | API 接口定义 |
| `ruoyi-ui/src/views/` | 页面组件 |

### 2.4 后端关键目录

| 目录路径 | 作用 |
|---------|------|
| `ruoyi-modules/ruoyi-system/src/main/java/com/ruoyi/system/controller/` | 控制器（定义接口）|
| `ruoyi-modules/ruoyi-system/src/main/java/com/ruoyi/system/service/` | 服务层（业务逻辑）|
| `ruoyi-modules/ruoyi-system/src/main/java/com/ruoyi/system/mapper/` | 数据访问层（SQL 操作）|
| `ruoyi-modules/ruoyi-system/src/main/java/com/ruoyi/system/domain/` | 实体类（数据库表映射）|
| `ruoyi-modules/ruoyi-system/src/main/resources/mapper/` | MyBatis XML 文件（写 SQL）|

## 三、开发时需要修改的文件

### 3.1 添加新功能时

**后端需要修改：**
1. `ruoyi-modules/ruoyi-system/src/main/java/com/ruoyi/system/controller/` - 新建控制器
2. `ruoyi-modules/ruoyi-system/src/main/java/com/ruoyi/system/service/` - 新建服务接口和实现
3. `ruoyi-modules/ruoyi-system/src/main/java/com/ruoyi/system/mapper/` - 新建 Mapper
4. `ruoyi-modules/ruoyi-system/src/main/java/com/ruoyi/system/domain/` - 新建实体类
5. `ruoyi-modules/ruoyi-system/src/main/resources/mapper/` - 新建 MyBatis XML

**前端需要修改：**
1. `ruoyi-ui/src/api/` - 新建 API 接口定义
2. `ruoyi-ui/src/views/` - 新建页面组件
3. `ruoyi-ui/src/router/index.js` - 添加路由配置

### 3.2 修改配置时

| 修改内容 | 文件位置 |
|---------|---------|
| 后端接口地址 | `ruoyi-ui/.env.development` |
| 前端端口 | `ruoyi-ui/vue.config.js` |
| 数据库连接 | Nacos 配置中心（application-dev.yml）|
| Redis 连接 | Nacos 配置中心（application-dev.yml）|

## 四、模块依赖关系

```
ruoyi-gateway (网关)
    ↓ 调用
ruoyi-auth (认证中心) + ruoyi-modules/* (业务模块)
    ↓ 依赖
ruoyi-common/* (通用模块)
    ↓ 依赖
ruoyi-api/* (接口定义)
```

## 五、快速定位指南

### 我想修改用户登录逻辑

**后端：**
- `ruoyi-auth/src/main/java/com/ruoyi/auth/controller/TokenController.java`
- `ruoyi-auth/src/main/java/com/ruoyi/auth/service/`

**前端：**
- `ruoyi-ui/src/views/login.vue`
- `ruoyi-ui/src/store/modules/user.js`

### 我想添加一个新的管理页面

**后端：**
1. 在 `ruoyi-modules/ruoyi-system/src/main/java/com/ruoyi/system/controller/` 新建控制器
2. 在 `ruoyi-modules/ruoyi-system/src/main/java/com/ruoyi/system/service/` 新建服务
3. 在 `ruoyi-modules/ruoyi-system/src/main/java/com/ruoyi/system/mapper/` 新建 Mapper
4. 在 `ruoyi-modules/ruoyi-system/src/main/java/com/ruoyi/system/domain/` 新建实体类
5. 在 `ruoyi-modules/ruoyi-system/src/main/resources/mapper/` 新建 MyBatis XML

**前端：**
1. 在 `ruoyi-ui/src/api/` 新建 API 文件
2. 在 `ruoyi-ui/src/views/` 新建页面组件
3. 在 `ruoyi-ui/src/router/index.js` 添加路由

### 我想修改菜单权限

- `ruoyi-modules/ruoyi-system/src/main/java/com/ruoyi/system/service/impl/SysMenuServiceImpl.java`
- 数据库表: `sys_menu`

### 我想修改接口权限

- `ruoyi-common/ruoyi-common-security/src/main/java/com/ruoyi/common/security/annotation/`
- Nacos 配置: `application-dev.yml` 中的 `ignore.urls` 配置

## 六、文件命名规范

### 后端

- Controller: `XxxController.java`
- Service 接口: `IXxxService.java`
- Service 实现: `XxxServiceImpl.java`
- Mapper: `XxxMapper.java`
- Mapper XML: `XxxMapper.xml`
- 实体类: `Xxx.java`（对应数据库表 `xxx`）

### 前端

- 页面组件: `xxx.vue`
- API 文件: `xxx.js`
- 路由名称: 与页面路径对应
