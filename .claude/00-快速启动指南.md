# 若依微服务项目 - 快速启动指南

> 傻瓜式启动文档，跟着步骤做即可！

## 一、启动前准备（必做！）

### 1.1 检查你的环境

你需要安装这些软件，**缺一不可**：

```bash
# 检查 Java 版本（必须是 17+）
java -version
# 应该显示: openjdk version "17.x.x" 或更高

# 检查 Maven
mvn -version
# 应该显示: Apache Maven 3.x.x

# 检查 Node.js
node -v
# 应该显示: v14.x - v16.x

# 检查 MySQL
mysql --version
# 应该显示: mysql  Ver 5.7.x 或 8.0.x

# 检查 Redis
redis-cli --version
# 应该显示: redis-cli 3.x.x 或更高
```

**如果上面任何一个命令报错，请先安装对应的软件！**

### 1.2 下载并启动 Nacos（最重要！）

**Nacos 是微服务的大脑，必须先启动它！**

```bash
# 如果还没有 Nacos，先下载
# https://github.com/alibaba/nacos/releases
# 下载 nacos-server-3.x.zip

# 解压后进入目录
cd /你的路径/nacos/bin

# 启动 Nacos（单机模式）
./startup.sh -m standalone

# Windows 用户用这个
startup.cmd -m standalone
```

**验证 Nacos 是否启动成功：**
- 打开浏览器访问: http://localhost:8848/nacos
- 用户名: `nacos`
- 密码: `nacos`

### 1.3 启动 Redis

```bash
# macOS (Homebrew)
brew services start redis

# Linux
redis-server

# Windows
# 直接运行 redis-server.exe
```

### 1.4 启动 MySQL

```bash
# macOS
brew services start mysql

# Linux
sudo systemctl start mysql

# Windows
# 在服务管理器中启动 MySQL 服务
```

### 1.5 初始化数据库

```bash
# 登录 MySQL
mysql -u root -p

# 创建数据库
CREATE DATABASE `ry-cloud` DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci;

# 退出 MySQL
exit;

# 导入 SQL 文件（按顺序）
cd /Users/cheang/Downloads/auvin/work/maintain/admin/auvin_learn_full_stack
mysql -u root -p ry-cloud < sql/ry_20250523.sql
```

### 1.6 配置 Nacos（重要！）

**这一步必须做，否则后端服务会启动失败！**

1. 访问 http://localhost:8848/nacos
2. 点击「配置管理」→「配置列表」
3. 点击右上角「导入配置」
4. 选择项目里的 `sql/ry_config_20260311.sql` 文件
5. 点击「导入」

## 五、常见问题

### ❌ 问题 1: 启动报错 "Connection refused"

**原因:** Nacos 没启动

**解决:** 先启动 Nacos！

```bash
cd /你的nacos路径/bin
./startup.sh -m standalone
```

### ❌ 问题 2: 端口已被占用

**原因:** 端口被其他程序占用

**解决:** 找到占用端口的程序并关闭

```bash
# 查看占用端口的程序
lsof -i :8080  # Gateway 端口
lsof -i :9200  # Auth 端口
lsof -i :1024  # 前端端口

# 杀掉进程
kill -9 <进程ID>
```

### ❌ 问题 3: 前端无法连接后端

**原因:** 后端服务没启动或端口不对

**解决:**
1. 检查后端服务是否启动
2. 检查 `ruoyi-ui/.env.development` 中的配置是否正确

### ❌ 问题 4: 数据库连接失败

**原因:** MySQL 没启动或数据库没创建

**解决:**
```bash
# 启动 MySQL
brew services start mysql  # macOS
sudo systemctl start mysql  # Linux

# 检查数据库是否存在
mysql -u root -p -e "SHOW DATABASES LIKE 'ry-cloud';"
```

### ❌ 问题 5: Redis 连接失败

**原因:** Redis 没启动

**解决:**
```bash
# 启动 Redis
brew services start redis  # macOS
redis-server  # Linux
```

### ❌ 问题 6: Nacos 配置导入失败

**原因:** 配置文件太大或格式有问题

**解决:**
1. 检查 `sql/ry_config_20260311.sql` 文件是否存在
2. 手动在 Nacos 界面添加配置（参考文档中的配置说明）

## 六、停止服务

### 停止后端服务

在每个终端窗口按 `Ctrl + C`

### 停止前端

在前端终端窗口按 `Ctrl + C`

### 停止 Nacos

```bash
cd /你的nacos路径/bin
./shutdown.sh
```

### 停止 Redis 和 MySQL

```bash
# macOS
brew services stop redis
brew services stop mysql

# Linux
sudo systemctl stop redis
sudo systemctl stop mysql
```
